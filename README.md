# CDA

## Development

### Setup

```markdown
$ npm install -g @nestjs/cli
$ npm install -g concurrently
$ npm install

$ npm run docker:build
```

### Start 

```markdown
# Docker
$ npm run docker:up

# Nest
$ npm run start
```

### Test

```markdown
# Unit
$ npm run test

# Functional
$ npm run test:functional
```

## Production

### Build

```markdown
$ npm run build
```
